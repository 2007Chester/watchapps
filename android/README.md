# WatchApps Android

Android‑приложение **WatchApps**.

## Назначение

- Авторизация пользователя
- Просмотр каталога циферблатов и приложений
- Покупка цифровых товаров (через backend)
- Загрузка APK с сервера
- Передача APK на часы с Wear OS
- Получение статуса установки от часов

## Технологии (план)

- Kotlin
- Android Jetpack (ViewModel, Navigation, etc.)
- Retrofit/OkHttp для API
- Jetpack Compose или XML (решить позже)
- Google Play Services / Wear OS Data Layer
