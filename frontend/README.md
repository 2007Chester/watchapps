# WatchApps Frontend

Фронтенд сайта **watchapps.ru**.

## Назначение

- Публичный каталог циферблатов и приложений для Wear OS
- Страница товара
- Личный кабинет пользователя
- Авторизация через backend WatchApps
- Интеграция с российскими способами оплаты (через backend)
- SEO‑оптимизированная витрина под русскоязычную аудиторию

## Технологии (план)

- Next.js 15 (App Router)
- TypeScript
- React
- TailwindCSS
- Axios или fetch для REST API
- SSR/SSG для SEO

Более детально см. ARCHITECTURE.md и ROADMAP.md.
