# Roadmap фронтенда WatchApps

## Этап 1 — Базовый каркас

- Инициализация Next.js + TS + Tailwind
- Базовый layout и тема
- Шапка, футер, навигация

## Этап 2 — Каталог

- Главная страница
- Страницы: top/new/discounts
- Страница категории
- Страница товара (без реальных платежей)

## Этап 3 — Авторизация и профиль

- Логин/регистрация через backend API
- Личный кабинет
- Список купленных циферблатов

## Этап 4 — Интеграция с платежами

- Отображение статуса оплаты
- Страница “после покупки”

## Этап 5 — Оптимизация и SEO

- SSR/SSG для ключевых страниц
- Микроразметка
